## CSS

un noeud de la page est affiché en fonction de 3 pricpales propriétés :
- margin = marges
- le boader
- le padding

exemple de CSS :
```css
div{
    margin:16px; /* marges de 16 pixels, haut, bas, gauche, droite */
    margin-top: 32px; /* override la marge du haut à 32 pixel */
    border: solid 8px purple;
    pading: 24px;
}
```
